# Agentic Data Steward

O que faz: Automação inteligente que recebe planilhas e PDFs "sujos" e usa IA para limpar, categorizar e salvar em banco de dados.

Tecnologias: React, TypeScript, Supabase e OpenAI/Gemini.

Diferencial: Focado em reduzir custos operacionais de pequenas empresas através de limpeza de dados automatizada.
